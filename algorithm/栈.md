# 栈

## 栈的定义

栈（stack）是限定仅在表尾进行插入和删除操作的线性表。
我们把允许插入和删除的一端成为栈顶（top），另一端称为栈底，不包含任何数据元素的栈称为空栈。栈又称为后进先出的线性表，简称 LIFO 结构

## 栈的抽象数据类型

```md
ADT 栈（Stack）
DATA
同线性表
Operation
push(e) 把元素 e 推到栈中
pop() 把 top 上的元素弹出，并返回
length() 返回 stack 的个数
peek() 返回当前栈顶的元素
```

## 栈的顺序存储

```js
class Stack {
  constructor() {
    this.dataStore = [];
    this.top = 0;
  }
  push(e) {
    this.dataStore.push(e);
    this.top++;
  }
  pop() {
    let item = this.dataStore.pop();
    this.top--;
    return item;
  }
  clear() {
    this.top = 0;
    this.dataStore = [];
  }
  peek() {
    return this.dataStore[this.top - 1];
  }
  length() {
    return this.top;
  }
  toString() {
    console.log(`当前栈的内容为：${JSON.stringify(this.dataStore)}`);
  }
}
```

## 栈的链式存储

```js
/**
 * 栈的数据类型
 */
class Stack {
  constructor() {
    this.top = null; //用于存储当前top信息
    this.size = 0;
  }
  push(e) {
    const item = new StackNode(e);
    item.next = this.top;
    this.top = item;
    this.size++;
  }
  pop() {
    const item = this.top;
    this.top = this.top.next;
    this.size--;
    return item.element;
  }
  peek() {
    return this.top.element;
  }
  clear() {
    this.top = null;
    this.size = 0;
  }
  toString() {
    console.log(`当前栈的内容为：${JSON.stringify(this.top)}`);
  }
}
```
